# Backend

## Технологии
- Node.js
- Express.js
- MongoDB.

## Структура проекта

- `configs/` - конфигурационные файлы
- `controllers/` - контроллеры для обработки запросов
- `middlewares/` - промежуточное ПО
- `models/` - модели данных MongoDB
- `routes/` - маршруты API
- `utils/` - вспомогательные функции

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер:

```bash
npm run dev
```
